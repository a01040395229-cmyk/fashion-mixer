# Style Rewinder - 주요 기능 명세서 (AI 기능 제외)

이 문서는 **Style Rewinder (3D Fashion Mixer)** 프로젝트의 핵심 기능과 구현 방식을 정리한 문서입니다. (요청에 따라 AI 생성 및 분석 관련 기능은 제외되었습니다.)

---

## 1. 3D 인터랙티브 인터페이스 (Three.js)
* **5단 독립 실린더 구조**: 패션 카테고리(`HAT`, `ACC`, `TOP`, `BOTTOM`, `SHOES`)에 맞춰 총 5개의 3D 원기둥(Cylinder)이 수직으로 배치되어 독립적으로 회전합니다.
* **마우스/터치 드래그 회전**: 사용자가 마우스나 터치 패드로 특정 실린더 영역을 드래그하여 아이템을 자유롭게 믹스매치할 수 있습니다. 마우스에서 손을 떼면 가장 가까운 칸(정렬 위치)으로 스냅(Snap) 이동합니다.
* **자율 회전 및 제어 복원**: 사용자 조작이 없을 때는 실린더들이 무작위 방향과 속도로 미세하게 자동 회전(Auto-rotation)하며, 조작이 발생하는 즉시 자율 회전이 일시 중지(5~10초)된 후 서서히 복원됩니다.
* **포커싱 셰이더 효과 (Custom Shader)**: 
  * 화면 중앙(정면)에 위치한 아이템은 뚜렷하게 보이도록 처리하고, 옆면으로 돌아가는 아이템은 서서히 흐려지거나(Blur) 어두워지도록(Darken) 커스텀 물리 셰이더(`MeshPhysicalMaterial`)가 적용되어 시각적 몰입감을 높입니다.

---

## 2. 패션 아이템 및 카테고리 구조
* **카테고리 구성**: 
  1. `HAT` (모자 - 높이 8.5)
  2. `ACC` (액세서리 - 높이 5.0)
  3. `TOP` (상의 - 높이 18.0)
  4. `BOTTOM` (하의 - 높이 20.0)
  5. `SHOES` (신발 - 높이 7.0)
* **동적 텍스처 합성 (Canvas API)**: 
  * 카테고리별로 업로드된 이미지들을 Canvas API를 이용해 가로형 파노라마 이미지(최대 너비 8,192px)로 즉석에서 이어붙이고, 이를 3D 실린더의 매핑 텍스처(`CanvasTexture`)로 변환합니다.

---

## 3. 스타일 세트(스타일링 프리셋) 캐러셀
* **무한 루프 테마 캐러셀**: 상단에 배치된 스타일 세트 목록은 좌우 무한 스크롤이 가능하도록 삼중 배열 형태로 렌더링되며, 현재 활성화된 세트에 맞춰 페이지네이션 도트가 동기화됩니다.
* **프리셋 저장 및 원클릭 적용**: 
  * `SAVE`: 현재 실린더가 가리키고 있는 5가지 아이템 조합의 회전 각도를 저장하여 특정 스타일 세트에 등록합니다.
  * `APPLY`: 스타일 세트를 선택하면 5개의 실린더가 해당 세트에 저장된 아이템 정렬 각도로 부드럽고 정확하게 자동 회전합니다.
* **대표 룩북 이미지 연동**: 각 스타일 세트마다 오피셜 룩북 이미지를 업로드할 수 있으며, 활성화된 세트의 스타일 가이드를 팝업 창을 통해 확인할 수 있습니다.

---

## 4. 갤러리 및 데이터 관리 패널 (`GALLERY`)
* **초고밀도 이미지 업로드**: 
  * 웹 로컬 스토리지의 한정된 용량(5MB) 문제를 극복하기 위해, 업로드 시 이미지 크기를 최대 720px로 조절하고 화질을 70% 압축하는 **자동 리사이징 필터**를 탑재하여 최대 100장의 이미지를 안정적으로 저장할 수 있습니다.
* **드래그 앤 드롭 아이템 순서 변경**: 갤러리 썸네일 그리드 내에서 아이템을 마우스로 끌어다 놓아 직관적으로 전시 순서를 재배치할 수 있으며, 스크롤 위치 보존 기술을 적용하여 편집 편의성을 확보했습니다.
* **아이템 상세 정보 바인딩**:
  * 각 이미지 아래의 입력창을 통해 아이템 명칭(Name), 상세 설명(Desc), 그리고 구매 혹은 랜딩 페이지로 이동 가능한 외부 주소(URL Link)를 직접 수정하고 저장할 수 있습니다.
* **세트 소속 드롭다운**: 각 아이템이 어떤 스타일 세트에 포함될 것인지 다중 선택 가능한 인라인 체크박스 메뉴를 제공합니다.

---

## 5. 부가 편의 기능
* **셔플(SHUFFLE) 기능**: 현재 실린더의 정렬 스냅 상태를 유지하면서 모든 실린더를 무작위 칸수만큼 부드럽게 고속 회전시켜 뜻밖의 조합을 찾아내도록 돕습니다.
* **전시용 단독 파일 패키징 (`SHARE AS .html`)**:
  * 현재 데이터베이스에 업로드된 모든 이미지와 세트 정보, 현재 정렬 각도를 정적으로 변환하여 HTML 소스 코드 내부(`window.EMBEDDED_DATA`)에 직접 주입합니다.
  * 이로써 별도의 서버 없이 파일 하나만 전달해도 똑같은 상태로 구동할 수 있는 100% 독립형 오프라인 전시 파일을 내보낼 수 있습니다.
* **비공개 전시 모드 잠금 (`Shift + L`)**:
  * 단축키 `Shift + L`을 통해 일반 관람객이 편집할 수 없도록 저장 및 편집 로직을 제한하는 '전시 모드'와 '편집 모드'를 토글할 수 있습니다.
